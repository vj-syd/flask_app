A Dream Journey

We had a dream. To make a cloud project. Together with a bunch of talented people. We came together on this journey of 2 weeks.

This was the first time! Excitement was high. We eagerly looked forward to the awesome project.

People had joined in from Sydney andMelbourne. There were five of us . Nobody knew how to start. But, deep within we were all Elon Musk- creator, entrepreneur. We believed we could find the way. We talked, scoured the internet for ideas, took inspiration from our hearts. Finally, we found the idea.

One that was close to our heart. We wanted to create a fun, story writing project. It would be a story with blanks left for users to fill in. Thus, every user would make their unique story. The project would display textbeautifully on the screen, and data analytics would search for some customized images from the internet. What a beautiful customer experience it would be. A fun outing for children of all ages, 5 to 105. That was the dream.

We planned to use python, css and html to create the project.

Running the project in an Agile way. It was all new, by the way.

Each one set out to do their part. Progress was very slow. Time was running low. Deadlines to be caught. So many pieces of the puzzle were unknown. We knew we could be lost. We kept trying every day. Little by Little. Slowly the project developed in parts. Then, came theflood! Animals, birds, food or body parts- all could be seen on screen!

We were finally all so happy. So excited. A dream journey was accomplished.
